# US Retail Price Tracker
## Project Overview
This project collects product pricing data from major US retailers using web scraping and analyzes pricing trends.
## Business Problem
Retailers frequently change product prices due to promotions, demand, and inventory levels. 
Manually tracking pricing across multiple retailers is time-consuming and inefficient.

This project builds an automated tool to monitor retail pricing and generate insights for market analysis.
## Data Collection
The project collects product pricing data from major US retailers such as:

- Amazon
- Best Buy
- Newegg
## Tools
- Python
- BeautifulSoup
- Pandas
- Matplotlib
## Workflow
1. Scrape product data from retail websites
2. Clean and structure the dataset
3. Analyze pricing trends
4. Visualize price differences across retailers
## Example Insight
Initial analysis shows that certain retailers offer frequent short-term promotions that significantly affect average pricing.
